Ebola on Ethereum

The ebola genome on the Ethereum blockchain served up from a smart contract.

Smart Contract Ethereum Address: 0x0

Check out the Github Repo for all the code. Or read the Medium Article about the the why and how it was created.

The contract has a few functions you can use each will produce a responce. The genome file is hosted on Swarm, Ethereum's decentrailzed file storage. A bit.ly link will redirect you to the full Swarm URL (its kind of long).

Interacting with the Smart Contract:

IMPORANT: Sending Ether or tokens to this contract is a bad idea. You will not get these funds back, and there is no method to retrive them if you do so. If you want to tip, use the tipping address (0x9Ff24857b990a8d036D0973d9a0825A76C2c8723).

Functions that can be used with Web3.js and Node.js or Geth and thier restulting reponces from the smart contract.


Callable Functions:

"getEbola()"

Returns a bit.ly URL that will redirect to the smart contract

{{> getEbola}}

"getInfo()"

Returns two strings. "0" Is a hello message from the creator in Esperanto. "1" Is some basic info on the ebola virus genome the contract contains a link to; sourced from here.

{{> getInfo}}

"tipCreator()"

Returns a text string and an Ethereum address. "0" Is a message from the creator in English. "1" Is the address you can send a tip to if you're feeling generous and or amused.

{{> tipCreator}}

"kill()"

Returns a text string. While you can try to execute this function with a transaction, it will not do anything as only the creator can kill the contract. If crete a transaction calling this function and you are not the creator, the transaction will still process, but the contact will live on... You'd be wasting your Ether and time.
